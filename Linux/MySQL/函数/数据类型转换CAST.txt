函数：CAST(value AS type)

作用：将 value 数据转换为 type 数据类型。
	数据类型 type：
		1.CHAR[(N)]	字符型
		2.DATE	日期型
		3.DATETIME	日期和时间型
		4.DECIMAL	float型
		5.SIGNED	int型
		6.TIME	时间型

例：
	mysql> select CAST('2019-01-23 11:35:48' AS CHAR);
	+-------------------------------------+
	| CAST('2019-01-23 11:35:48' AS CHAR) |
	+-------------------------------------+
	| 2019-01-23 11:35:48                 |
	+-------------------------------------+

	mysql> select CAST('2019-01-23 11:35:48' AS DATE);
	+-------------------------------------+
	| CAST('2019-01-23 11:35:48' AS DATE) |
	+-------------------------------------+
	| 2019-01-23                          |
	+-------------------------------------+

	mysql> select CAST('2019-01-23 11:35:48' AS DATETIME);
	+-----------------------------------------+
	| CAST('2019-01-23 11:35:48' AS DATETIME) |
	+-----------------------------------------+
	| 2019-01-23 11:35:48                     |
	+-----------------------------------------+

	mysql> select CAST('2019-01-23 11:35:48' AS SIGNED);
	+---------------------------------------+
	| CAST('2019-01-23 11:35:48' AS SIGNED) |
	+---------------------------------------+
	|                                  2019 |
	+---------------------------------------+

	mysql> select CAST('2019-01-23 11:35:48' AS TIME);
	+-------------------------------------+
	| CAST('2019-01-23 11:35:48' AS TIME) |
	+-------------------------------------+
	| 11:35:48                            |
	+-------------------------------------+

	mysql> select CAST('2343.31431' AS DECIMAL(10,2));
	+-------------------------------------+
	| CAST('2343.31431' AS DECIMAL(10,2)) |
	+-------------------------------------+
	|                             2343.31 |
	+-------------------------------------+
