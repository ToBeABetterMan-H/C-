概念
{
	1.Decimal 为 SQL Serve、MySQL 等数据库的一种数据类型，不属于浮点数类型。
	2.可以在定义时划定整数部分以及小数部分的位数。
	3.使用精确小数类型不仅能够保证数据计算更为精确，还可以节省存储空间，例如百分比使用decimal(4,2)即可。
}

例
{
	1.decimal(10,2)中的"2"表示小数部分的位数，10指的是整数部分加小数部分的总长度。
	2.如果插入的值未指定小数部分或者小数部分不足"2"位则自动补到"2"位小数。
	3.如果插入的值小数部分超过了"2"位则只截取前"2"为小数。
	4.插入的数字整数部分不能超过"10"-"2"位，否则不能插入成功，会报超出范围的错误。
}
